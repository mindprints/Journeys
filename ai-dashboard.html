Image Generation
Used as the last-resort image fallback when Wikimedia, Openverse and Brave find nothing.
Must support modalities: ["image"] on OpenRouter. Recommended: google/gemini-3.1-flash-image-preview.

The server automatically chooses a prompt style based on the Wikipedia description of the topic:

Person (e.g. "American computer scientist") Portrait illustration in a professional setting relevant to their field. Editorial style, no text.
Place (e.g. "city in France") Evocative landscape or cityscape, clean modern illustration. No text or labels.
Physical object / organism Technical cutaway or isometric illustration with labelled components. Scientific style.
Abstract concept / algorithm / process ← default for AI topics Educational infographic diagram — labeled boxes, arrows, data-flow, architecture layers. Looks like a textbook figure or IEEE paper diagram. Not decorative art.
